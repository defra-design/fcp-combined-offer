Select and apply rotational actions to your land

Move a rotational action to a new land parcel

  1. Use the map or land parcel link to select the land the action is currently applied to.
  2. Deselect the action, then select ‘Save’.
  3. Select the new land parcel you want it to apply to.
  4. Select the action.
  5. Enter the quantity of the action (area size, meterage or number) using 4 decimal places, then select ‘Save’.

Update a rotational action

Use the map or land parcel link to select the land the action is currently applied to, then either:

  • deselect the action to remove it entirely
  • enter the new quantity of the action (area size, meterage or number) using 4 decimal places
